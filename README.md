# APEX11

APEX11 is a modern web application for a football kit and jersey e-commerce storefront.

## Tech Stack

- **Framework**: Next.js 15 (App Router)
- **UI & Logic**: React 19
- **Animations**: Framer Motion
- **Styles**: Vanilla CSS

---

## Project Structure

- `app/` — Next.js application router pages, layouts, and components.
- `public/` — Static assets (images, icons).
- Root HTML/CSS/JS (`index.html`, `shop.html`, `styles.css`, `script.js`) — Original static prototype files for reference.

---

## Getting Started

### 1. Install Dependencies
```bash
npm install
```

### 2. Run the Development Server
```bash
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

### 3. Build for Production
```bash
npm run build
```
